# 🧠 프로젝트 브레인: 대시보드 기능정의

> **최종 업데이트**: 2026-03-18  
> **관련 파일**: `dashboard.html`

---

## 1. 대시보드 개요

통합관리자가 시스템 전반의 운영 현황을 한눈에 파악하기 위한 메인 화면.

---

## 2. 구성 요소

### 2.1 기간 필터 바
| 옵션 | 비교 대상 |
|------|-----------|
| 오늘 | 어제 대비 |
| 어제 | 그 전날 대비 |
| 최근 7일 | 이전 7일 대비 |
| 최근 30일 | 이전 30일 대비 |

### 2.2 Today KPI 카드 (5개)
| # | 항목 | 아이콘 | 색상 | 데이터 예시 |
|---|------|--------|------|-------------|
| 1 | 오늘 활성 기관 수 | 🏫 | 파랑 | 54 / 105 |
| 2 | 오늘 활성 교사 수 | 👩‍🏫 | 초록 | 156 / 162 |
| 3 | 오늘 활성 학부모 수 | 👨‍👩‍👧 | 보라 | 88 / 195 |
| 4 | 오늘 접속자 수 | 📱 | 시안 | 312 |
| 5 | 오늘 게시글 수 | 📝 | 황색 | 55 |

- 각 카드에 **전일 대비 증감 트렌드** 표시 (↑ 초록, ↓ 빨강)

### 2.3 활성 사용자 추이 차트 (DAU/MAU)
- **DAU**: 일간활성 사용자 (전체, 교사, 학부모 3개 라인)
- **MAU**: 월간활성 사용자 (전체 1개 라인)
- Chart.js Line 차트

### 2.4 오늘의 운영 To-do
| 항목 | 설명 | 색상 |
|------|------|------|
| 미답변 Q&A | 답변 대기 중 문의 | 빨강 |
| 기관 가입 승인 요청 | 승인 대기 기관 | 황색 |
- 클릭 시 해당 관리 화면으로 이동

### 2.5 메뉴별 사용량 차트
- **게시글 수** 또는 **사용 기관 수** 전환
- 메뉴: 알림장, 공지사항, 앨범, 식단표, 투약의뢰, 출석부, 스케줄
- Chart.js Bar 차트

### 2.6 공지사항
- **최신순** / **조회순** 정렬 전환
- 태그: ALL, 기관, 학부모, 교사
- 공지 데이터: 제목, 날짜, 조회수

### 2.7 기관/교사 활동 랭킹
- **기관 기준** / **교사 기준** 전환
- TOP 5, LOW 5 양방향 표시
- 지표: 게시글 수 / 열람률(%)

### 2.8 디바이스별 이용 분포
- Chart.js Doughnut 차트
- Android 52.1%, iOS 22.8%, Xiaomi 13.9%, Other 11.2%

## 3. 대시보드 데이터 구조

```javascript
// DAU 데이터
DAU_DATA = {
  labels: ['날짜 배열 (3/4~3/30)'],
  total: [Number],    // 전체 접속자
  teacher: [Number],  // 교사
  parent: [Number]    // 학부모
}

// MAU 데이터
MAU_DATA = {
  labels: ['Jan'~'Jun'],
  total: [Number]
}

// 메뉴별 게시글 수
MENU_POSTS = { labels: [메뉴명], data: [Number] }

// 메뉴별 사용 기관 수
MENU_ORGS = { labels: [메뉴명], data: [Number] }

// 랭킹 데이터
RANK_*_TOP/LOW = [{ name, posts, pct }]

// 공지사항
NOTICES = [{ tag, tagClass, text, date, views }]

// 디바이스
DEVICE_DATA = [{ name, pct, color }]
```
