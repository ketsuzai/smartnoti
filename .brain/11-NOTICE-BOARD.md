## 1. 알림장 개요

### 1.1 개념 정의

- **알림장(Notice Board)**은 교사 ↔ 학부모 간 일상 커뮤니케이션 채널이다.
- **보낸 알림장**: 교사가 작성하여 학부모에게 전달하는 알림장 (기존 구조 유지)
- **받은 알림장**: 학부모가 교사에게 작성하여 보낸 상황 전달 메시지 (신규 구조)
- 두 방향의 알림장은 **받은 알림장 / 보낸 알림장 탭**으로 화면에서 구분한다.

### 1.2 비즈니스 규칙 요약

```
보낸 알림장 (notice_boards) ──1:N──→ notice_children ──1:N──→ notice_comments
받은 알림장 (parent_notices) ──N:1──→ children · 학부모 단방향 메시지
```

| 규칙 | 내용 |
|------|------|
| 보낸 알림장 | 교사 1회 작성 → 반 전체 또는 선택 원아에게 동시 발행 |
| 받은 알림장 | 학부모가 특정 카테고리(결석/건강/일정/준비물)로 교사에게 전달 |
| 댓글 격리 | `notice_child_id` 기준 — 원아 간 댓글 노출 불가 |
| 수정 범위 | 교사: 본인 작성 알림장 수정 / 기관관리자: 전체 수정 |
| 삭제 | 교사: 본인만 / 기관관리자 이상: 모두 |
| 발행 상태 | `draft` → `published` → `recalled` |
| AI 기능 | 예문 자동생성, 문장 다듬기(교정) |
| 종합 평가 | 교사 작성 시 6개 카테고리 3단계 평가, 학부모 화면에 요약 표시 |

> ⚠️ **원아별 보기(child view)는 제거됨** — 받은 알림장 / 보낸 알림장 탭 구조로 대체

## 4. 교사 종합 평가 (daily_eval) 설계

### 4.1 개요

- **위치**: 알림장 작성 패널 → 원아별 카드(write-child-card) 하단 섹션
- **목적**: 오늘 하루 원아 상태를 6개 카테고리로 빠르게 요약하여 학부모에게 전달
- **저장**: ✔ 저장 클릭 시 → `daily_eval_summary` 자동 생성 + 원아 카드 상단 배지 즉시 반영
- **표시**: 상세 뷰어 원아 탭 상단에 종합 평가 카드(6칸 그리드)로 표시

### 4.2 6개 카테고리 (확장 UI 상세)

| 카테고리 | 필드 | 옵션 | 확장 표시 조건 | 확장 내용 |
|---------|------|------|:---:|---------|
| 기분 | `daily_eval_mood` | 좋음/보통/나쁨 | — | — |
| 건강 | `daily_eval_health` | 좋음/보통/나쁨 | **나쁨** 선택 시 | 체온 입력 + 증상 칩(복수) + 추가 메모 |
| 식사량 | `daily_eval_meal` | 좋음/보통/나쁨 | **나쁨** 선택 시 | 추가 메모 textarea |
| 배변 | `daily_eval_toilet` | 좋음/보통/나쁨 | **항상** (조건 없음) | 배변 상태(단일) + 횟수 입력 + 메모 |
| 참여도 | `daily_eval_activity` | 좋음/보통/나쁨 | **선택 시** (3가지 모두) | 메모 textarea (placeholder 상태별 분기) |
| 교우관계 | `daily_eval_social` | 좋음/보통/나쁨 | **선택 시** (3가지 모두) | 관찰 메모 textarea |

**건강 증상 칩**: 발열 / 기침 / 콧물 / 설사 / 구토 / 복통 / 두통 / 알레르기

**배변 상태 옵션**: 정상 / 딱딱함 / 묽음 / 설사 / 배변없음

**참여도 placeholder 분기**:
- 좋음 → "적극적으로 참여한 활동을 기록해주세요"
- 보통 → "참여도 특이사항을 기록해주세요"
- 나쁨 → "참여를 어려워한 이유나 상황을 기록해주세요"

### 4.3 요약 생성 규칙

```
좋음 ≥ 4개 & 나쁨 = 0개 → "아주 좋아요 👍"
나쁨 ≥ 1개              → "확인이 필요해요 ⚠️"
그 외                   → "무난해요 😊"
```

### 4.4 정책

- 전체 선택 필수 아님 (일부만 선택 가능)
- 건강 나쁨 → 체온/증상 1개 이상 권장 (프로토타입에서 필수 아님)
- 학부모 조회 화면: **읽기 전용** (수정 불가)
- 초기값: 미선택 / 초기화 버튼 제공
