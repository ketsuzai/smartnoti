## 7. 콘텐츠 관리 필터링 정책

### 7.1 페이지 통합 원칙
- **기관관리자 + 교사관리자**는 동일한 HTML 파일(`org-dashboard.html` 등) 사용
- `sessionStorage.userRole`로 권한 분기 처리
- **통합관리자가 운영관리로 진입한 경우**도 `org-dashboard.html` 재사용 (UI 동일)
- 통합관리자의 추가 권한(계정 정지, 기관 수정 등)은 **통합관리자 전용 페이지에서만** 행사

### 7.2 반 필터 UI 정책 (상단 셀렉트박스)

| 역할 | 표시 범위 | 선택 가능 여부 |
|------|----------|---------------|
| 통합관리자 (운영관리 컨텍스트) | 기관 전체 반 | 모든 반 선택 가능 |
| 기관관리자 | 기관 전체 반 | 모든 반 선택 가능 |
| 교사관리자 | 기관 전체 반 표시 | **담당 반만 선택 가능**, 비담당 반은 회색 처리 + disabled |

### 7.3 교사관리자 반 필터 상세 규칙

- 셀렉트박스에 **기관 전체 반 목록이 표시**됨 (존재 인식 가능)
- 담당 반: 정상 텍스트, 선택 가능
- 비담당 반: 회색 처리(`color: #9CA3AF`), `disabled` 속성 적용
- 담당 반이 **1개**인 경우: 해당 반 자동 선택 (변경 불가)
- 담당 반이 **여러 개**인 경우: 첫 번째 담당 반 기본 선택, 담당 반 간 전환 가능
- `sessionStorage.assignedClasses` 배열로 담당 반 목록 관리

```javascript
// 교사관리자 담당 반 세션 구조
sessionStorage.setItem('assignedClasses', JSON.stringify([
  { id: 'CLS001', name: '햇님반', age: '만3세', studentCount: 18 },
  { id: 'CLS003', name: '별빛반', age: '만5세', studentCount: 22 } // 복수 담당 예시
]));
sessionStorage.setItem('currentClass', 'CLS001'); // 현재 선택된 반
```

### 7.4 기관관리자의 반 필터링

- 기관관리자는 해당 기관의 **모든 반**이 보이며 모두 선택 가능
- 알림장, 원아관리, 앨범 등 콘텐츠 접근 시 **반 필터링 UI 필수 제공**
- 기본 진입 시: '전체 반' 선택 상태로 시작

## 9. 운영관리 — 통합관리자 전용 기능 상세

통합관리자가 운영관리 메뉴 접근 시:

1. **기관 선택 팝업** 노출 (SearchableSelector 형태)
2. 기관 선택 완료 → 해당 기관의 권한 컨텍스트로 전환
3. **세션은 통합관리자 유지**, 권한 컨텍스트만 해당 기관으로 조정
4. 상단바에 현재 접속 기관명 표시 (예: "🏫 해맑은 어린이집 권한으로 접속 중")
5. 기관 재선택 또는 운영관리 컨텍스트 종료 버튼 제공

### 기관 선택 팝업 스펙
- 새 창(팝업 모달) 형태로 열림
- 기관 카드 그리드 + 검색 입력 (기관명, 원장이름)
- 카드에 표시할 정보: **기관명, 원장이름, 주소, 전화번호**
- 기관 선택 → "확인" 버튼 → 해당 기관 권한으로 진입
- 기관 검색: 실시간 필터링 (SearchableSelector 패턴)
