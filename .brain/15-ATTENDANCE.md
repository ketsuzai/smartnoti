- **역할**: 원아의 일별/월별 등원·하원·결석·인정결석을 반 단위로 관리
- **권한**: org_admin → 전체 반 + 전체보기; teacher → 담당 반만
- **UI 구조**: 반 필터(대시보드 디자인) → 뷰탭(일별/월별) → KPI 4개 → 일별 테이블 or 월별 그리드 → 우측 드로어
- **핵심 상태**: 등원(boolean) + 하원(boolean) + 결석(status) + 인정결석(status)

## 1. 상태 정의

### 1.1 원자 필드

| 필드 | 타입 | 설명 |
|------|------|------|
| `arrived` | boolean | 원아 등원 여부 |
| `arrived_at` | string (HH:MM) | 등원 시각 |
| `departed` | boolean | 원아 하원 여부 |
| `departed_at` | string (HH:MM) | 하원 시각 |
| `absence_type` | null \| 'absent' \| 'approved' | 결석 유형 |
| `absence_reason` | string | 결석/인정결석 사유 (인정결석 시 필수) |
| `note` | string | 교사 메모 |

### 1.2 파생 표시 상태 (UI)

| 조건 | 표시 상태 | 배지 색상 |
|------|-----------|-----------|
| arrived=true, departed=true | **하원완료** | blue |
| arrived=true, departed=false | **등원** | green |
| absence_type='absent' | **결석** | red |
| absence_type='approved' | **인정결석** | amber |
| arrived=false, absence_type=null | **미기록** | gray |

### 1.3 상태 전환 규칙

- 등원=true로 변경 시 → 결석/인정결석 자동 해제
- 등원=false로 변경 시 → 하원 비활성
- 인정결석 선택 시 → 사유 필수

## 4. 화면 구성

### 4.1 반 필터 (상단)

- 대시보드(`operation-dashboard.html`)와 동일한 `.class-filter-wrap` 디자인 사용
- `org_admin`: 전체 반 옵션 포함 / `teacher`: 담당 반만 활성
- KPI 카드 + 테이블이 반 선택에 따라 실시간 갱신

### 4.2 뷰 탭

| 탭 | 설명 |
|----|------|
| 일별 현황 | 날짜 선택 → 해당 날 원아별 출결 테이블 |
| 월별 출석부 | 월 선택 → 원아 × 날짜 격자 그리드 |

### 4.3 날짜 네비

- ◀ / ▶ 버튼으로 전일/익일 이동
- [오늘] 버튼으로 즉시 이동
- 월별 탭에서는 ◀ / ▶ 로 월 이동

### 4.4 KPI 카드 (4개, 반별 갱신)

| 카드 | 아이콘 | 색상 | 내용 |
|------|--------|------|------|
| 전체 원아 | 👶 | blue | 선택 반(또는 전체) 원아 수 |
| 등원 | ✅ | green | 등원 원아 수 + 출석률 프로그레스 바 |
| 결석 | ❌ | red | 결석 원아 수 |
| 인정결석 | 📝 | amber | 인정결석 원아 수 |

### 4.5 일별 현황 테이블

| 컬럼 | 설명 |
|------|------|
| 번호 | 순번 |
| 이름 | 아바타 + 이름 |
| 성별 | 남/여 |
| 반 | 전체 뷰일 때만 표시 (색상 태그) |
| 등원 시각 | 등원 시각 (green) / 미등원 시 — |
| 하원 시각 | 하원 시각 (blue) / 미하원 시 — |
| 상태 | 상태 배지 |
| 사유/메모 | 결석 사유 또는 교사 메모 |
| 수정 | 드로어 오픈 버튼 |

### 4.6 월별 출석부 그리드

- 행: 원아 (이름 컬럼 sticky)
- 열: 해당 월의 날짜 (주말 회색 배경)
- 셀 아이콘:
  - `●` (dep, blue) = 하원완료
  - `○` (arr, green) = 등원
  - `✕` (abs, red) = 결석
  - `△` (appr, amber) = 인정결석
  - `·` (none, gray) = 미기록
  - 빈 칸 = 주말/공휴일

- 전체 반 선택 시 반별 섹션 헤더로 구분
- 가로 스크롤 지원

### 4.7 출결 수정 드로어 (우측)

**헤더**: 반명 · 원아명 · 날짜

**섹션 1 — 등원**:
- 등원 여부 토글 (green)
- 등원 시각 time input (disabled when off)

**섹션 2 — 하원** (등원=true일 때 활성):
- 하원 여부 토글 (blue)
- 하원 시각 time input (disabled when off)

**섹션 3 — 결석 처리** (등원=false일 때 활성):
- 결석 / 인정결석 / 미기록 라디오 버튼
- 사유 textarea (인정결석 선택 시 필수)

**섹션 4 — 메모**: 선택 입력

**CTA**: 취소 / 저장
