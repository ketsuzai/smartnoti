## 3. 공통 규칙 요약

### 3.1 담당 반 범위 적용 페이지

교사관리자가 **담당 반 범위로 제한**되는 페이지:

| 페이지 | 제한 기준 |
|--------|---------|
| 원아 관리 | 담당 반 소속 원아 |
| 알림장 | 담당 반 대상 알림장 |
| 공지사항 | 담당 반 대상 공지 |
| 앨범 | 담당 반 앨범 |
| 투약의뢰서 | 담당 반 원아 의뢰서 |
| 반 관리 | 담당 반 수정만 |

> `sessionStorage.assignedClasses` 배열로 담당 반 목록 관리.
> 담당 반 1개: 자동 선택 / 여러 개: 첫 번째 반 기본 선택 후 전환 가능.

### 3.2 수정 이력 기록 대상

기관관리자가 **타인 작성 콘텐츠를 수정/삭제**할 경우 이력 기록:

| 페이지 | 이력 기록 조건 |
|--------|-------------|
| 기관정보 | 항상 기록 |
| 알림장 | 기관관리자가 타 교사 알림장 수정·삭제 시 |

### 3.3 예외 규칙

| 페이지 | 예외 내용 |
|--------|---------|
| 일정 관리 | 교사도 기관 전체 일정 생성·수정·삭제 가능 (담당 반 제한 없음) |
| 초대 관리 | 교사가 타 교사 발송 초대장 조회 가능 (읽기 전용) |
